    如何参与

    运行

    我们建议使用 github 进行协作open in new window。

    • 点击 fock
    • clone 代码到本地

    准备环境

    1. 准备 node 环境和依赖：
    • 安装 nodeopen in new window
    • 安装 pnpm
    npm i -g pnpm@8.12.1 --registry=https://registry.npmmirror.com
    1. 下载 main.exeopen in new window 到根目录。

    安装依赖和运行项目

    # 安装根目录依赖
pnpm i

# 安装文档安依赖
cd doc
pnpm i

# 在根目录运行项目
cd ..
pnpm run dev
